üöÄ How to Run Locally

Option 1: Streamlit Web App

  1. Clone the repository: git clone https://github.com/Biswajitn23/asl-recognition-system
  2. Install dependencies: pip install -r requirements.txt
  3. Run the app: streamlit run app.py
  4. Open browser: http://localhost:8501

Option 2: Flask Web App

  1. Clone the repository
  2. Install dependencies: pip install -r requirements.txt
  3. Run the app: python flask_app.py
  4. Open browser: http://localhost:5000

Option 3: Desktop App with TTS

  1. Clone the repository
  2. Install dependencies
  3. Run: python real_time.py
  4. Uses text-to-speech for recognized signs

üìö Training Process

The model was trained using a custom pipeline that extracts hand landmarks from YouTube ASL tutorial videos:

  • Data Collection: Downloaded 6 professional ASL tutorial videos
  • Frame Extraction: Extracted 500 frames per video (3,000 total)
  • Hand Landmark Detection: Used MediaPipe to detect 21 hand landmarks in 3D space
  • Feature Engineering: Normalized landmarks relative to wrist position
  • Model Training: Random Forest classifier with 200 estimators
  • Sign Distribution: 25 different signs per video (147 unique signs total)

Train Your Own Model

Run: python youtube_asl_trainer.py

Customize the YouTube URLs in the script to train on different signs!
